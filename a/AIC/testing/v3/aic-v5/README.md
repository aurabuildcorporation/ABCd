# AIC-V3 — Adaptive Intelligence Core (Version 3)

AIC-V3 is a hybrid Flask + Node system that performs entity scoring, enrichment, anomaly detection, and historical trend tracking.

## Architecture

### Flask Frontend
- Bootstrap UI
- Chart.js visualizations
- Entity search and scoring
- History dashboards

### Node Scoring Engine
- Embeddings
- Sentiment analysis
- Heuristics
- Volatility scoring
- Yelp Fusion enrichment
- Weighted fusion scoring

### Python Workers
- Interval scoring
- Trend tracking
- Anomaly detection
- Forecasting stubs

### Database
- SQLite
- Entities, scores, history, logs
