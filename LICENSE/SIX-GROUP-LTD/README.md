# AIC Network

**Operated by SIX GROUP LTD.**

AIC (Artificial Intelligence Credit) is a network-based intelligence system that evaluates participating entities using continuous data-driven scoring models.

## Overview

The AIC Network is composed of three core layers:

### 1. AIC Score System
A dynamic, algorithmically generated metric representing relative performance and behavioral positioning across network participants.

### 2. Internal Credit System
A closed-loop unit of account used within the AIC Network for access, exchange, and participation in ecosystem services.

### 3. Network Infrastructure
A continuously evolving system for data ingestion, evaluation, and economic interaction between participants.

---

## Key Properties

- Continuous evaluation model
- Dynamic scoring system
- Internal credit-based ecosystem
- Modular governance structure
- Operator-controlled evolution (SIX GROUP LTD.)

## Important Notice

AIC is an analytical and network-based system. It does not provide financial advice or guarantees of economic outcome.

Internal credits are not fiat currency and do not represent bank deposits or legal tender.
